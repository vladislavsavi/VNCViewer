# electron-VNCViewer

VNC-client

для работы нужен vnc server (протестировано с x11vnc)

# Начало работы
### Установка зависимостей
  npm
```
nmp i
```
  yarn
```
yarn
```

# Команды

  Запуск приложения для разработки (включает в себя запуск webpack и electron)
```
npm run dev
```
```
yarn dev
```

### Сборка установщика
  Сначала нужно собрать react приложение
```
npm run build
```
```
yarn build
```
  Собираем установщик
```
npm run dist
```
```
yarn dist
```
